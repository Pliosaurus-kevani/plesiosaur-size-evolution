# Results of fitting Brownian motion and Ornstein-Uhlenbeck process to three plesiosaur subsets

Model fitting was accomplished in [RevBayes](https://revbayes.github.io/) 1.3.2. The results can be read and processed in the [R script](https://github.com/Pliosaurus-kevani/plesiosaur-size-evolution/tree/main/R%20scripts) **BM_OU.R**.
