# Results of variable-rates model fitted to the neck principal coordinate of plesiosaurs

Model fitting was performed in [BayesTraits](https://www.evolution.reading.ac.uk/BayesTraitsV5.0.3/BayesTraitsV5.0.3.html) 4.0.0. The results can be read and processed in the [R script](https://github.com/Pliosaurus-kevani/plesiosaur-size-evolution/tree/main/R%20scripts) **neck_rate_mass.R**.
