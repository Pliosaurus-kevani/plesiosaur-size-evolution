# phylogenetic analysis: partitioned

Phylogenetic analysis was run on [RevBayes](https://revbayes.github.io/) 1.3.2

* **appendicular_ordered.nex**: ordered appendicular characters
* **appendicular_unordered.nex**: unordered appendicular characters
* **axial_ordered.nex**: ordered axial characters
* **axial_unordered.nex**: unordered axial characters
* **craniodental_ordered.nex**: ordered craniodental characters
* **craniodental_unordered.nex**: unordered craniodental characters
* **range.txt**: stratigraphic information. All ages were shifted forward by 66 million years
* **SFBD_lognormal_partitioned.Rev**: RevBayes script of the phylogenetic analysis
