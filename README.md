# Constrained mass evolution and decoupled morphological rates in plesiosaurs

This repository contains data and commented code to enable readers to reproduce the analyses conducted in Zhao and Zhang, 2026.

A brief introduction to each folder is presented here. Detailed introduction to each file can be found in the **README.md** file in the corresponding folder.

**raw data**:
>Raw skeletal measurements and corresponding references.

**data for R**:
>Processed data for analyses in [R](https://www.r-project.org/).

**R scripts**:
>[R](https://www.r-project.org/) scripts to reproduce the results and figures presented in the manuscript.

**phylo_unpartitioned**:
>Data for the unpartitioned analysis performed in [RevBayes](https://revbayes.github.io/).

**phylo_partitioned**:
>Data for the partitioned analysis performed in [RevBayes](https://revbayes.github.io/).

**BM_OU**:
>[RevBayes](https://revbayes.github.io/) scripts and data files for the comparison of Brownian motion versus Ornstein-Uhlenbeck process fitted to plesiosaur body mass subsets.

**BM_OU_output**:
>[RevBayes](https://revbayes.github.io/) results for the comparison of Brownian motion versus Ornstein-Uhlenbeck process fitted to plesiosaur body mass subsets.

**BayesTraits_output**:
>Results of the variable-rates fitted to plesiosaur neck data using [BayesTraits](https://www.evolution.reading.ac.uk/BayesTraitsV5.0.3/BayesTraitsV5.0.3.html).
