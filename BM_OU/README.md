# Model fitting and comparison of Brownian motion and Ornstein-Uhlenbeck process based on the three subsets

Scripts were run on [RevBayes](https://revbayes.github.io/)  1.3.2.

* **mass_Elas.nex**: body mass data file for Elasmosauridae
* **trees_Elas.nex**: 100 randomly selected posterior trees for Elasmosauridae
*  **Elasmosauridae_OU_BM.Rev**: RevBayes script to perform model fitting and comparison
* **mass_Tha.nex**: body mass data file for Thalassophonea
* **trees_Tha.nex**: 100 randomly selected posterior trees for Thalassophonea
*  **Thalassophonea_OU_BM.Rev**: RevBayes script to perform model fitting and comparison
* **mass_others.nex**: body mass data file for other plesiosaurs
* **trees_others.nex**: 100 randomly selected posterior trees for other plesiosaurs
*  **others_OU_BM.Rev**: RevBayes script to perform model fitting and comparison
