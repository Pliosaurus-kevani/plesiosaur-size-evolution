# phylogenetic analysis: unpartitioned
Phylogenetic analysis was run on [RevBayes](https://revbayes.github.io/) 1.3.2.
* **ordered.nex**: ordered characters
* **unordered.nex**: unordered characters
* **range.txt**: stratigraphic information. All ages were shifted forward by 66 million years.
* **SFBD_lognormal_ordered.Rev**: RevBayes script of the phylogenetic analysis.
